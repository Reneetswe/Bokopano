# Bokopano - Work. Grow. Belong.

A clean, modern volunteer and work-exchange platform connecting passionate individuals with meaningful opportunities across Africa.

## Project Overview

Bokopano is a platform designed to facilitate skill exchanges between volunteers and hosts across African countries. Users can trade their skills and expertise for accommodation, meals, and unique cultural experiences.

## Features

- **Modern, Clean Design**: Professional interface with no emojis or clutter
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Opportunity Listings**: Browse and discover volunteer opportunities across Africa
- **Contact System**: Integrated contact form for inquiries
- **Smooth Animations**: Subtle animations and transitions for better user experience
- **Accessibility**: Built with accessibility best practices

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern CSS with custom properties, Grid, and Flexbox
- **Vanilla JavaScript**: No dependencies, pure JavaScript functionality
- **Google Fonts**: Playfair Display and DM Sans for typography

## Customization

### Colors
Edit the CSS variables in `:root` at the top of `styles.css`:

```css
:root {
  --clay: #C4622D;
  --earth: #2A1F14;
  /* ... other variables */
}
```

### Fonts
Update font imports in the HTML `<head>` section and CSS font-family declarations.

### Content
Edit HTML content directly in `index.html`. All sections are clearly commented for easy identification.
